# Privacy Policy

## Collection of Personal Data

We collect the following personal data:

- Name
- Email address

## Use of Personal Data

The personal data collected is used for:

- Responding to user requests
- Improving our website
- Sending newsletters and marketing communications

## Data Protection

We take the security of personal data seriously and implement security measures to protect this data against unauthorized access or disclosure.

## Data Retention

We retain the collected personal data for as long as necessary to achieve the purposes mentioned above or as long as required by law.

## User Rights

Users have the following rights regarding their personal data:

- Right of access
- Right to rectification
- Right to erasure
- Right to restrict processing
- Right to data portability

## Cookie Policy

We use cookies on our website. Cookies are small text files that are stored on the user's device by the web browser and allow us to recognize the user when they return to the website.
